--> Overview
This repository contains my practice and learning materials for R programming and data visualization using ggplot2. 
It includes basic to intermediate examples of data handling, statistical plotting, and visualization techniques.

--> Topics Covered
R basics (data frames, vectors, functions)

Data manipulation

ggplot2 visualization

Different types of plots:

📈 Scatter Plot (geom_point())

📉 Line Plot (geom_line())

📊 Bar Chart (geom_bar())

📦 Histogram (geom_histogram())

🥧 Pie Chart (coord_polar())
